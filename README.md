# Docker-compose tip

This is a presentation done for knowledge sharing in the workplace.

It's based on Reveal.js, so it can be started simply with:
```
git clone https://github.com/ProvoK/docker-compose-tips compose-tips
cd compose-tips
npm i
npm start
```
